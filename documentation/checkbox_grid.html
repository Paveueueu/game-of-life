checkbox_grid module¶

class checkbox_grid.CheckboxGrid(master=None, **kwargs)¶

Bases: CTkFrame

Grid of checkboxes for selecting rules. Allows the user to set how many neighbors make a cell is born (Spawn rules) and with how many neighbors a cell should die (Die rules).

Attributes:

checkboxes_spawn (list): list of checkboxes for spawn rules checkboxes_die (list): list of checkboxes for die rules labels (list): list of labels for the grid

Methods:

get_rules: returns lists of set spawn and die rules
